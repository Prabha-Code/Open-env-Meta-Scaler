🧠 AI Customer Support Intelligence Environment
Overview
A real-world environment simulating customer support workflows where an AI agent must analyze, classify, prioritize, and resolve customer tickets.

Features
Multi-dimensional decision-making
Sentiment + category + priority detection
Reward shaping for partial correctness
Progressive difficulty tasks
Tasks
Easy → simple issues
Medium → ambiguous problems
Hard → real-world complex cases
Reward System
Category: 0.25
Priority: 0.25
Sentiment: 0.2
Action: 0.3
Use Case
Training and evaluating LLM agents for real-world support automation.

Check out the configuration reference at https://huggingface.co/docs/hub/spaces-config-reference
